# Multi-Step Signup Wizard Documentation

Complete documentation for the multi-step clinic registration wizard with email verification and Stripe payment integration.

## Overview

The multi-step signup wizard replaces the single-page registration form with an improved user experience across 5 distinct steps, making the registration process more intuitive and less overwhelming.

## Features

- Modern design with gradient backgrounds
- Progress tracking with visual indicators
- Real-time form validation
- Email verification system
- Stripe payment integration
- Responsive design (mobile, tablet, desktop)
- Auto-save functionality
- Password strength indicator
- Subdomain availability checker

## Quick Start

1. Install dependencies: `npm install`
2. Configure environment variables in `.env`
3. Run database migration
4. Start server: `npm start`
5. Navigate to `/register-wizard.html`

## Configuration

See `.env.example` for required environment variables including:
- SMTP settings for email
- Stripe keys for payments
- Application URL for links
