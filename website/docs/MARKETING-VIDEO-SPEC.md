## Exporting 2K–8K

The component is **HTML/CSS**, not a video file. To get broadcast-resolution masters:

1. **Browser capture**  
   - Open `/reel` on a large monitor.  
   - Set zoom so the framed area maps to your target width (e.g. 1440×2560 for 2.5K vertical).  
   - Use OBS, Chrome DevTools device mode with custom device size, or a dedicated recorder at **high bitrate**.

2. **Figma / After Effects**  
   - Use this storyboard as a **shot list**: three beats + logo lockup.  
   - Recreate motion in Figma prototypes or AE for full control and **ProRes / 8K** export.

3. **Remotion / Lottie**  
   - Port timing and copy into Remotion for programmatic 4K/8K renders (future iteration).
